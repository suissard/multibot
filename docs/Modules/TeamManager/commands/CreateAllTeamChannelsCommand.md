# `CreateAllTeamChannelsCommand`

Lance la création/synchronisation de tous les salons d\'équipe pour un segment donné.

## Arguments

| Name | Type | Description | Required |
| ---- | ---- | ----------- | -------- |
| `segment` | `STRING` | Le nom du segment à synchroniser (Optionnel). | No |

# `notifyMatchCommand`

Exécute la commande pour envoyer une notification de match à partir d'un ID.

**Parameters:**

| Name | Type | Description |
| ---- | ---- | ----------- |
| `args` | `object` | - Les arguments de la commande. |

**Returns:** `Promise<string>` - message de confirmation ou d'erreur.

# `ErrorEvent`

## Description

Affiche les erreurs de discord.js.

Gère l'événement 'error' de discord.js. Affiche l'erreur dans la console.

**Parameters:**

| Name | Type | Description |
| ---- | ---- | ----------- |
| `error` | `Error` | - L'objet Erreur fourni par l'événement. |

# `DebugEvent`

Gère l'événement 'debug' de discord.js. Affiche les informations de débogage dans la console.

**Parameters:**

| Name | Type | Description |
| ---- | ---- | ----------- |
| `debug` | `string` | - L'information de débogage fournie par l'événement. |
